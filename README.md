# skincare-server

## Getting started

- Clone repository (SSH)

```
git clone git@github.com:ngocngo-agilityio/nextjs-practice-e-commerce-server.git

```

- Change the path to the folder you just cloned to your computer

```
cd nextjs-practice-e-commerce-server

```

- Checkout branch

```
git checkout main

```

- Install packages

```
pnpm install

```

- Run server

```
pnpm run server

```
